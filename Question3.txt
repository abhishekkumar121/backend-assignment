B)How do you list out the hidden directories using Linux command line tool?
Ans)To list out the hidden directories using a Linux command line tool, you have a few options:
1)Using the "ls" command with the "-a" option:
    -->The "-a" option shows all files and folders, including hidden ones
    -->Command: `ls -a`
2)Using the "ls" command with the "-A" option:
    -->The "-A" option is similar to "-a" but excludes the implied files
    -->Command: `ls -A`
3)Using the "find" command:
    -->To show hidden directories in the current working directory without recursive search, use the "find" command with the "-type d" option
    -->Command: `find . -type d -name ".*"`
